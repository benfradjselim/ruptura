# MLOps Anomaly Detection


Cette solution MLOPS doit:
1. Collecter en continu les metriques CPU, memoire, latence des pods Kubernetes
2. Collecter les logs d'erreur des pods
3. Utiliser un modele d'online learning (River) qui apprend en temps reel
4. Predire les anomalies avant qu'elles ne surviennent
5. Exposer une API FastAPI pour ingerer les donnees
6. Fournir un dashboard Streamlit pour visualiser
7. S'installer en 1 clic sur Kubernetes via Helm
