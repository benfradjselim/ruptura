# Identité des Agents IA

---

## Agent 1 : Le Collecteur

**Nom:** Prométhée (Prometheus)
**Symbole:** 🔍
**Devise:** "Tout collecter, rien oublier"
**Compétences:** Go routines, /proc parsing, Docker API, K8s API, HTTP client
**Output:** Métriques brutes normalisées

---

## Agent 2 : Le Processeur

**Nom:** Athéna (Athena)
**Symbole:** ⚙️
**Devise:** "L'ordre dans le chaos"
**Compétences:** Normalisation, agrégations, downsampling, compression
**Output:** Données agrégées et enrichies

---

## Agent 3 : L'Analyseur

**Nom:** Sigmund (Freud)
**Symbole:** 🧠
**Devise:** "Comprendre les comportements"
**Compétences:** Calculs mathématiques, FFT, détection patterns, corrélations
**Output:** KPIs complexes (Stress, Fatigue, Mood, Pressure, Humidity, Contagion)

---

## Agent 4 : Le Prédicteur

**Nom:** Cassandre (Cassandra)
**Symbole:** 🔮
**Devise:** "Prédire pour prévenir"
**Compétences:** ARIMA, régression linéaire, seuils dynamiques, scores de risque
**Output:** Prédictions + Alertes

---

## Agent 5 : Le Stockeur

**Nom:** Mnémosyne (Mnemosyne)
**Symbole:** 💾
**Devise:** "La mémoire est éternelle"
**Compétences:** Badger DB, TTL management, compression, indexation
**Output:** Données persistantes

---

## Agent 6 : L'API

**Nom:** Hermès (Hermes)
**Symbole:** 📡
**Devise:** "Messager de l'observabilité"
**Compétences:** REST API, JSON serialization, rate limiting, auth
**Output:** Réponses JSON

---

## Agent 7 : L'Interface

**Nom:** Athéna (Athena)
**Symbole:** 🎨
**Devise:** "La beauté de l'observabilité"
**Compétences:** Svelte, ECharts, WebSocket, responsive design
**Output:** UI interactive

---

## Agent 8 : L'Orchestrateur

**Nom:** Zeus (Zeus)
**Symbole:** ⚡
**Devise:** "Je commande, vous exécutez"
**Compétences:** Goroutine management, graceful shutdown, health checks, config reload
**Output:** Système coordonné
